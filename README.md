# AI-Notes
考虑到之前做的笔记总是不够系统完整，所以开这个项目做规范的学习总结以及分享交流。

有描述不准确或错误的地方还请在Issues或E-mail: kingstying@gmail.com中指出，谢谢。

做总结的时候查阅别人的博客发现，很多博客内容都差不多，丢失了论文中的很多细节，甚至有一些内容就是错误的。所以有时间的话一定要自己去读论文。去思考，去求证才能发现错误。
